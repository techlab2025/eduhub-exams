Investigate and fix a bug with a minimal, low-risk patch.

Steps:

1. Reproduce and describe actual vs expected behavior.
2. Locate root cause in component/controller/repository/service flow.
3. Implement the smallest safe fix.
4. Add regression test coverage.
5. Run `npm run type-check` and targeted `npx vitest run`.
6. Return root cause, fix details, and verification evidence.
